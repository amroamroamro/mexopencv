cv.GraphSegmentation/GraphSegmentation

Creates a graph based segmentor

obj = cv.GraphSegmentation()
obj = cv.GraphSegmentation('OptionName',optionValue, ...)

Options

  • Sigma The sigma parameter, used to smooth image. default 0.5
  • K The k parameter of the algorithm. default 300
  • MinSize The minimum size of segments. default 100
