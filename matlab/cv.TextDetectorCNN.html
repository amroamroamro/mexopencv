cv.TextDetectorCNN

Class providing functionality of text detection

This class provides the functionality of text bounding box detection (finds bounding boxes of text words given an input image).

This class uses OpenCV dnn module to load pre-trained model described in [LiaoSBWL17]. The original repository with the modified SSD Caffe version: GitHub.

Model (90.68 MB) can be downloaded from DropBox.

Modified .prototxt file with the model description can be obtained from GitHub.

References

[LiaoSBWL17]:

Minghui Liao, Baoguang Shi, Xiang Bai, Xinggang Wang, and Wenyu Liu. "Textboxes: A fast text detector with a single deep neural network". In AAAI, 2017.
