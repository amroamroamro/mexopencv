save (cv.LBPHFaceRecognizer) - MATLAB File Help
cv.LBPHFaceRecognizer/save

Saves a FaceRecognizer and its model state

obj.save(filename)
str = obj.save(filename)

Input

  • filename The filename to store this FaceRecognizer to (either XML/YAML).

Output

  • str optional output. If requested, the model is persisted to a string in memory instead of writing to disk.

Saves this model to a given filename, either as XML or YAML.

Saves the state of a model to the given filename.
