cv.RotationWarper/warp

Projects the image

[dst,tl] = obj.warp(src, K, R)
[dst,tl] = obj.warp(src, K, R, 'OptionName',optionValue, ...)

Input

  • src Source image.
  • K Camera intrinsic parameters, 3x3 single matrix.
  • R Camera rotation matrix, 3x3 single matrix.

Output

  • dst Projected image.
  • tl Project image top-left corner point [x,y].

Options

  • InterpMode Interpolation mode, see cv.remap. default 'Linear'
  • BorderMode Border extrapolation mode, see cv.remap. default 'Constant'
