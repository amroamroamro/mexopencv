cv.Estimator/estimate

Estimates camera parameters

cameras = obj.estimate(features, pairwise_matches)
[cameras, success] = obj.estimate(...)

Input

  • features Features of images. See cv.FeaturesFinder.
  • pairwise_matches Pairwise matches of images. See cv.FeaturesMatcher.

Output

  • cameras Estimated camera parameters. Structure that describes camera parameters with the following fields:
    • aspect Aspect ratio.
    • focal Focal length.
    • ppx Principal point X.
    • ppy Principal point Y.
    • R 3x3 camera rotation matrix.
    • t 3x1 camera translation vector.
    • K 3x3 camera intrinsic parameters.
  • success True in case of success, false otherwise.
