subspaceReconstruct (cv.LDA) - MATLAB File Help
cv.LDA.subspaceReconstruct

Reconstructs projections

dst = cv.LDA.subspaceReconstruct(W, mn, src)

Input

  • W eigenvectors.
  • mn mean vector.
  • src projected data.

Output

  • dst reconstructed data

Reconstructs data as X = Y*W' + mean.
