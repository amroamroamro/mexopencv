cv.dictionaryDump

Dump dictionary (aruco)

s = cv.dictionaryDump(dictionary)

Input

  • dictionary dictionary of markers.

Output

  • s Output struct with the following fields:
    • bytesList
    • markerSize
    • maxCorrectionBits
    • bits
