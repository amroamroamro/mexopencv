runByKeypointSize (cv.KeyPointsFilter) - MATLAB File Help
cv.KeyPointsFilter.runByKeypointSize

Remove keypoints of sizes out of range

keypoints = cv.KeyPointsFilter.runByKeypointSize(keypoints, minSize, maxSize)

Input

  • keypoints input collection of keypoints.
  • minSize minimum keypoint size.
  • maxSize maximum keypoint size. default realmax('single')

Output

  • keypoints output filtered keypoints.
