getObjectnessValues (cv.ObjectnessBING) - MATLAB File Help
cv.ObjectnessBING/getObjectnessValues

Return the list of the rectangles' objectness value

objectnessValues = obj.getObjectnessValues()

Output

  • objectnessValues vector of floats in the same order as objectnessBoundingBox returned by the algorithm (in computeSaliency function). The bigger value these scores are, it is more likely to be an object window.
