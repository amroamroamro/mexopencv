cv.FacemarkKazemi/getFaces

Detect faces using the custom detector

[faces, success] = obj.getFaces(img)

Input

  • img Input image.

Output

  • faces Detected faces {[x,y,w,h], ...}. Each face is stored as rect.
  • success success flag.
