calculateShift (cv.AlignMTB) - MATLAB File Help
cv.AlignMTB/calculateShift

Calculates shift between two images

shift = obj.calculateShift(img0, img1)

Input

  • img0 first image (uint8 grayscale).
  • img1 second image of same size and type as img0.

Output

  • shift calculated shift [x,y].

Calculates shift between two images, i. e. how to shift the second image to correspond it with the first.
