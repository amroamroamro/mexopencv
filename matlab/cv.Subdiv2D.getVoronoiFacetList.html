getVoronoiFacetList (cv.Subdiv2D) - MATLAB File Help
cv.Subdiv2D/getVoronoiFacetList

Returns a list of all Voroni facets

[facetList, facetCenters] = obj.getVoronoiFacetList(idx)

Input

  • idx Vector of vertices IDs to consider. For all vertices you can pass empty vector.

Output

  • facetList Output vector of the Voroni facets, an array of array of points {{[x,y], ...}, ...}.
  • facetCenters Output vector of the Voroni facets center points, a vector of points {[x,y], ...}.
