cv.BOWKMeansTrainer/add

Adds descriptors to a training set

trainer.add(descs)

Input

  • descs Descriptors to add to a training set. Each row of the descriptors matrix is a descriptor.

The training set is clustered using cluster method to construct the vocabulary.
