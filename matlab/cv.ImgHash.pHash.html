pHash (cv.ImgHash) - MATLAB File Help
cv.ImgHash.pHash

Computes pHash value of the input image

hash = cv.ImgHash.pHash(img)

Input

  • img input image want to compute hash value, type should be uint8 with 1/3/4 channels.

Output

  • hash Hash value of input, it will contain 8 uint8 values.
