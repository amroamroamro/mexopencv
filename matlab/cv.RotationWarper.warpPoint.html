cv.RotationWarper/warpPoint

Projects the image point

uv = obj.warpPoint(pt, K, R)

Input

  • pt Source point [x,y].
  • K Camera intrinsic parameters, 3x3 single matrix.
  • R Camera rotation matrix, 3x3 single matrix..

Output

  • uv Projected point [xx,yy].
