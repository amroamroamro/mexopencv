warpImage (cv.ShapeTransformer) - MATLAB File Help
cv.ShapeTransformer/warpImage

Apply a transformation, given a pre-estimated transformation parameters, to an Image

output = obj.warpImage(transformingImage)
output = obj.warpImage(..., 'OptionName',optionValue, ...)

Input

  • transformingImage Input image.

Output

  • output Output image.

Options

  • Interpolation Image interpolation method. default 'Linear'
  • BorderType border style. default 'Constant'
  • BorderValue border value. default 0

See cv.remap or cv.warpAffine for options description.
