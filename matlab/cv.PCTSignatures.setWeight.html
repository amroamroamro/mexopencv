setWeight (cv.PCTSignatures) - MATLAB File Help
cv.PCTSignatures/setWeight

Sets weight (multiplicative constant) that linearly stretch individual axis of the feature space

obj.setWeight(idx, value)

Input

  • idx ID of the weight (0-based index). One of:
    • 0 weight.
    • 1 X.
    • 2 Y.
    • 3 L.
    • 4 A.
    • 5 B.
    • 6 Contrast.
    • 7 Entropy.
  • value Value of the weight.
