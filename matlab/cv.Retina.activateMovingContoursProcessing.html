activateMovingContoursProcessing (cv.Retina) - MATLAB File Help
cv.Retina/activateMovingContoursProcessing

Activate/desactivate the Magnocellular pathway processing (motion information extraction)

obj.activateMovingContoursProcessing(activate)

Input

  • activate true if Magnocellular output should be activated, false if not. If activated, the Magnocellular output can be retrieved using the cv.Retina.getMagno method.

By default, it is activated.
