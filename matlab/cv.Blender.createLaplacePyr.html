cv.Blender.createLaplacePyr

Create Laplace pyramid

pyr = cv.Blender.createLaplacePyr(img, num_levels)
pyr = cv.Blender.createLaplacePyr(..., 'OptionName',optionValue, ...)

Input

  • img Input image.
  • num_levels Number of pyramid levels.

Output

  • pyr Laplace pyramid, cell-array of matrices.

Options

  • UseGPU (requires CUDA). default false
