cv.RidgeDetectionFilter/getRidgeFilteredImage

Apply Ridge detection filter on input image

out = obj.getRidgeFilteredImage(img)

Input

  • img Input array as supported by cv.Sobel. img can be 1-channel or 3-channels (color images are converted to grayscale).

Output

  • out Output array of depth as specified in DDepth in constructor. Output image with ridges.
