cv.WBDetector/read

Read detector from file

detector.read(filename)
detector.read(str, 'FromString',true)
detector.read(..., 'OptionName', optionValue, ...)

Input

  • filename Name of the file to read from.
  • str String containing serialized detector you want to load.

Options

  • FromString Logical flag to indicate whether the input is a a filename or a string containing the serialized detector. default false
