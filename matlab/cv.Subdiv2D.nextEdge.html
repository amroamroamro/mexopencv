cv.Subdiv2D/nextEdge

Returns next edge around the edge origin

e = obj.nextEdge(edge)

Input

  • edge Subdivision edge ID.

Output

  • e an integer which is next edge ID around the edge origin (eOnext on the picture shown if e is the input edge).
