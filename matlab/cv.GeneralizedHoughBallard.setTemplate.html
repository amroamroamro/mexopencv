setTemplate (cv.GeneralizedHoughBallard) - MATLAB File Help
cv.GeneralizedHoughBallard/setTemplate

Set template to search

hough.setTemplate(templ)
hough.setTemplate(edges, dx, dy)
hough.setTemplate(..., 'OptionName', optionValue, ...)

Input

  • templ template, 8-bit 1-channel image
  • edges template edges
  • dx template x-derivative of the same size as edges and single-precision floating type.
  • dy template y-derivate of the same size and type as dx.

Options

  • Center Template center [x,y]. The default [-1,-1] will use [size(templ,2) size(templ,1)]./2 as center.

In the first variant of the function (with the templ input), the edges are internally calculated using the cv.Canny filter, and the derivatives dx and dy using cv.Sobel operator.
