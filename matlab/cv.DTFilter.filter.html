cv.DTFilter/filter

Produce domain transform filtering operation on source image

dst = obj.filter(src)
dst = obj.filter(src, 'OptionName',optionValue, ...)

Input

  • src filtering image with unsigned 8-bit or floating-point 32-bit depth and up to 4 channels.

Output

  • dst destination image.

Options

  • DDepth optional depth of the output image. DDepth can be set to -1, which will be equivalent to class(src). default -1
