cv.KeyPointsFilter.convertFromPoints

Convert vector of points to vector of keypoints, where each keypoint is assigned the same size and the same orientation

keypoints = cv.KeyPointsFilter.convertFromPoints(points2f)
keypoints = cv.KeyPointsFilter.convertFromPoints(points2f, 'OptionName',optionValue, ...)

Input

  • points2f Array of (x,y) coordinates of each keypoint.

Output

  • keypoints Keypoints similar to those obtained from any feature detection algorithm like SIFT/SURF/ORB.

Options

  • Size keypoint diameter. default 1.0
  • Response keypoint detector response on the keypoint (that is, strength of the keypoint). default 1.0
  • Octave pyramid octave in which the keypoint has been detected. default 0
  • ClassId object id. default -1
