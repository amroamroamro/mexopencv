cv.LATCH/LATCH

Constructor

obj = cv.LATCH()
obj = cv.LATCH(..., 'OptionName',optionValue, ...)

Options

  • Bytes the size of the descriptor. Can be 64, 32, 16, 8, 4, 2 or 1. default 32
  • RotationInvariance whether or not the descriptor should compansate for orientation changes. default true
  • HalfSize the size of half of the mini-patches size. For example, if we would like to compare triplets of patches of size 7x7x then the half_ssd_size should be (7-1)/2 = 3. default 3
  • Sigma sigma value for cv.GaussianBlur smoothing of the source image. Source image will be used without smoothing in case sigma value is 0. default 2.0
