addLayerToPrev (cv.Net) - MATLAB File Help
cv.Net/addLayerToPrev

Adds new layer and connects its first input to the first output of previously added layer

id = net.addLayerToPrev(name, layerType, params)

Input

  • name unique name of the adding layer.
  • layerType typename of the adding layer (type must be registered).
  • params parameters which will be used to initialize the creating layer.

Output

  • id unique identifier of created layer, or -1 if a failure will happen.
