cv.drawMarkerAruco

Draw a canonical marker image

img = cv.drawMarkerAruco(dictionary, id, sidePixels)
img = cv.drawMarkerAruco(..., 'OptionName',optionValue, ...)

Input

  • dictionary dictionary of markers indicating the type of markers.
  • id identifier of the marker that will be returned. It has to be a valid id in the specified dictionary (0-based).
  • sidePixels size of the image in pixels.

Output

  • img output image with the marker, of size sidePixels-by-sidePixels.

Options

  • BorderBits width of the marker border. default 1

This function returns a marker image in its canonical form (i.e. ready to be printed).
