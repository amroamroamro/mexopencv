cv.TransientAreasSegmentationModule/write

Write xml/yml formatted parameters information

obj.write(fs)
str = obj.write(fs)

Input

  • fs the filename of the xml file that will be open and writen with formatted parameters information.

Output

  • str optional output. If requested, the parameters are persisted to a string in memory instead of writing to disk.
