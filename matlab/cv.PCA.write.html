write (cv.PCA) - MATLAB File Help
cv.PCA/write

Write PCA objects to file

obj.write(filename)
str = obj.write(filename)

Input

  • filename Name of the file to write to.

Output

  • str optional output. If requested, the object is persisted to a string in memory instead of writing to disk.

Writes eigenvalues, eigenvectors and mean to the specified file.
