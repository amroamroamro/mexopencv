cv.BRISK/detect

Detects keypoints in an image or image set

keypoints = obj.detect(img)
keypoints = obj.detect(imgs)
[...] = obj.detect(..., 'OptionName',optionValue, ...)

Input

  • img Image (first variant), 8-bit grayscale image.
  • imgs Image set (second variant), cell array of images.

Output

  • keypoints The detected keypoints. In the first variant, a 1-by-N structure array. In the second variant of the method, keypoints{i} is a set of keypoints detected in imgs{i}.

Options

  • Mask A mask specifying where to look for keypoints (optional). It must be a logical or 8-bit integer matrix with non-zero values in the region of interest. In the second variant, it is a cell-array of masks for each input image, masks{i} is a mask for imgs{i}. Not set by default.
