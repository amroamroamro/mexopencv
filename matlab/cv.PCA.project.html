cv.PCA/project

Projects vector(s) to the principal component subspace

Y = pca.project(X)

Input

  • X input vector(s); must have the same dimensionality and the same layout as the input data used at PCA phase, that is, if 'Row' was specified, then size(X,2)==size(data,2) (vector dimensionality) and size(X,1) is the number of vectors to project, and the same is true for the 'Col' case.

Output

  • Y output vectors (PC coefficients); in case of 'Col', the output matrix has as many columns as the number of input vectors, this means that size(Y,2)==size(X,2) and the number of rows match the number of principal components (for example, MaxComponents parameter passed to the constructor).

The method project one or more vectors to the principal component subspace, where each vector projection is represented by coefficients in the principal component basis.
