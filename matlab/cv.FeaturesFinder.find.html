cv.FeaturesFinder/find

Finds features in the given image

features = obj.find(img)
features = obj.find(img, rois)

Input

  • img Source image.
  • rois Regions of interest. A cell array of 4-element vectors {[x y w h], ...}

Output

  • features Found features. Structure containing image keypoints and descriptors with the following fields:
    • img_idx
    • img_size
    • keypoints
    • descriptors
