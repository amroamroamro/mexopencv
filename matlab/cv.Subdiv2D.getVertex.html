cv.Subdiv2D/getVertex

Returns vertex location from vertex ID

[pt, firstEdge] = obj.getVertex(vertex)

Input

  • vertex vertex ID.

Output

  • pt vertex [x,y].
  • firstEdge Optional. The first edge ID which is connected to the vertex.
