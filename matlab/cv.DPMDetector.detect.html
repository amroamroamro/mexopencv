cv.DPMDetector/detect

Detects objects

objects = detector.detect(img)

Input

  • img An image where objects are to be detected (8-bit integer or 64-bit floating-point color image).

Output

  • objects The detections. A struct array of detected objects with the following fields:
    • rect rectangle [x,y,w,h] of the object
    • score score of the detection
    • class name of the object class

Find rectangular regions in the given image that are likely to contain objects of loaded classes (models) and corresponding confidence levels.
