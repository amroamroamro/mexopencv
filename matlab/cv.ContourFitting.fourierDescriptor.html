cv.ContourFitting.fourierDescriptor

Fourier descriptors for planed closed curves

dst = cv.ContourFitting.fourierDescriptor(src)
dst = cv.ContourFitting.fourierDescriptor(src, 'OptionName',optionValue, ...)

Input

  • src input contour, vector of 2D points stored in numeric array Nx2/Nx1x2/1xNx2 or cell array of 2-element vectors {[x,y], ...}.

Output

  • dst 2-channel array of type single and length NumElt.

Options

  • NumElt number of rows in dst or cv.getOptimalDFTSize rows if NumElt=-1. default -1
  • NumFD number of FD to return in dst, dst = [FD(1...NumFD/2) FD(NumFD/2-NumElt+1...:NumElt)]. default -1 (return all of FD as is).

For more details about this implementation, please see [PersoonFu1977].
