cv.ContourFitting/ContourFitting

Create ContourFitting object

obj = cv.ContourFitting()
obj = cv.ContourFitting('OptionName',optionValue, ...)

Options

  • CtrSize number of contour points after resampling. default 1024
  • FDSize number of Fourier descriptors. default 16
