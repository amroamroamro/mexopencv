render (cv.Plot2d) - MATLAB File Help
cv.Plot2d/render

Renders the plot to a matrix

plotResult = obj.render()
plotResult = obj.render('OptionName',optionValue, ...)

Output

  • plotResult Plot result, 8-bit 3-channel image.

Options

  • FlipChannels whether to flip the order of color channels in output, from OpenCV's BGR to between MATLAB's RGB. default true
