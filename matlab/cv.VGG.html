cv.VGG

Class implementing VGG (Oxford Visual Geometry Group) descriptor

Trained end to end using "Descriptor Learning Using Convex Optimisation" (DLCO) aparatus described in [Simonyan14].

References

[Simonyan14]:

K. Simonyan, A. Vedaldi, and A. Zisserman. "Learning local feature descriptors using convex optimisation". IEEE Transactions on Pattern Analysis and Machine Intelligence, 2014.
