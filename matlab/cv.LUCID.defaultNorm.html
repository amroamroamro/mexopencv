defaultNorm (cv.LUCID) - MATLAB File Help
cv.LUCID/defaultNorm

Returns the default norm type

ntype = obj.defaultNorm()

Output

  • ntype Norm type. One of cv::NormTypes:
    • Inf
    • L1
    • L2
    • L2Sqr
    • Hamming
    • Hamming2

Always Hamming for LUCID.
