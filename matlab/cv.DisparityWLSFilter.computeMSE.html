cv.DisparityWLSFilter.computeMSE

Function for computing mean square error for disparity maps

mse = cv.DisparityWLSFilter.computeMSE(GT, src)
mse = cv.DisparityWLSFilter.computeMSE(..., 'OptionName',optionValue, ...)

Input

  • GT ground truth disparity map (int16 or single).
  • src disparity map to evaluate (int16 or single).

Output

  • mse returns mean square error between GT and src.

Options

  • ROI region of interest [x,y,w,h]. Not set by default.
