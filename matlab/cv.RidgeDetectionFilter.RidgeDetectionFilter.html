cv.RidgeDetectionFilter/RidgeDetectionFilter

Creates instance of the Ridge detection filter

obj = cv.RidgeDetectionFilter()
obj = cv.RidgeDetectionFilter('OptionName',optionValue, ...)

Options

  • DDepth Specifies output image depth, one of single or double. Default is single
  • Dx Order of derivative x, default is 1
  • Dy Order of derivative y, default is 1
  • KSize Sobel kernel size (1, 3, 5, or 7), default is 3
  • OutDType Converted format for output, default is uint8
  • Scale Optional scale value for derivative values, default is 1
  • Delta Optional bias added to output, default is 0
  • BorderType Pixel extrapolation method. See cv.copyMakeBorder. Default is 'Default'

Above options have the same meaning as cv.Sobel options.
