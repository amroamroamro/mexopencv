cv.HfsSegment/SlicSpixelSize

This parameter is used in the first stage mentioned above (the SLIC stage). It describes the size of each superpixel when initializing SLIC. Every superpixel approximately has slicSpixelSize * slicSpixelSize pixels in the begining.
