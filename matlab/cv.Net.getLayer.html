cv.Net/getLayer

Returns layer with specified id or name which the network use

layer = net.getLayer(layerId)

Input

  • layerId layer name or layer id.

Output

  • layer returned layer. Scalar structure with the following fields:
    • blobs List of stored learned parameters as returned by cv.Net.getParam.
    • name name of the layer instance, can be used for logging or other internal purposes.
    • type Type name which was used for creating layer by layer factory.
    • preferableTarget preferred target for layer forwarding (see cv.Net.setPreferableTarget).

Layers are the building blocks of networks.
