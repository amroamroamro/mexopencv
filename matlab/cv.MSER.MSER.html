cv.MSER/MSER

Full constructor for MSER detector

obj = cv.MSER()
obj = cv.MSER(..., 'OptionName',optionValue, ...)

Options

  • Delta delta, in the code, it compares (size_{i}-size_{i-delta})/size_{i-delta}. default 5.
  • MinArea prune the area which smaller than minArea. default 60.
  • MaxArea prune the area which bigger than maxArea. default 14400.
  • MaxVariation prune the area have similar size to its children. default 0.25
  • MinDiversity for color image, trace back to cut off mser with diversity less than MinDiversity. default 0.2.
  • MaxEvolution for color image, the evolution steps. default 200.
  • AreaThreshold for color image, the area threshold to cause re-initialize. default 1.01.
  • MinMargin for color image, ignore too small margin. default 0.003.
  • EdgeBlurSize for color image, the aperture size for edge blur. default 5.
