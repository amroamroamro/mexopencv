setMotionFilter (cv.OnePassStabilizer) - MATLAB File Help
cv.OnePassStabilizer/setMotionFilter

Set the motion filtering algorithm for the video stabilizer

stab.setMotionFilter(motionFilterType, 'OptionName',optionValue, ...)

Input

  • motionFilterType motion filtering method. One of:
    • GaussianMotionFilter

Options

  • Radius default 15
  • Stdev If negative, uses sqrt(Radius). default -1.0

The class uses GaussianMotionFilter by default.
