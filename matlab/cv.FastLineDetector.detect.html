cv.FastLineDetector/detect

Finds lines in the input image

lines = obj.detect(img)

Input

  • img A grayscale (uint8) input image.

Output

  • lines A cell array of 4-elements vectors of the form {[x1,y1, x2,y2], ..} specifying the beginning and ending point of a line, where point 1 [x1,y1] is the start, point 2 [x2,y2] the end. Returned lines are directed so that the brighter side is on their left.

An example output of the default parameters of the algorithm can be seen here:

If only a ROI needs to be selected, use:

lines = obj.detect(cv.Rect.crop(image, roi));
lines = cat(1, lines{:});
lines = bsxfun(@plus, lines, roi);
