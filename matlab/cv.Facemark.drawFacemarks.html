cv.Facemark.drawFacemarks

Utility to draw the detected facial landmark points

img = cv.Facemark.drawFacemarks(img, points)
img = cv.Facemark.drawFacemarks(..., 'OptionName',optionValue, ...)

Input

  • img The input image to be processed.
  • points Contains the data of points which will be drawn {[x,y], ...}.

Output

  • img Output image with drawn points.

Options

  • Color The color of points represented in BGR format, default [255,0,0].
