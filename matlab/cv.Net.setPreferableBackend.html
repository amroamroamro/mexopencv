cv.Net/setPreferableBackend

Ask network to use specific computation backend where it supported

net.setPreferableBackend(backend)

Input

  • backend computation backend supported by layers, one of:
    • Default
    • Halide Halide language backend.
    • InferenceEngine Intel's Deep Learning Inference Engine.
