cv.CascadeClassifier.convert

Convert classifier file from the old format to the new format

cv.CascadeClassifier.convert(oldcascade, newcascade)
status = cv.CascadeClassifier.convert(oldcascade, newcascade)

Input

  • oldcascade name of classifier file to read in old format
  • newcascade name of classifier file to write in new format

Output

  • status optional output success flag.
