cv.DisparityWLSFilter.readGT

Function for reading ground truth disparity maps

dst = cv.DisparityWLSFilter.readGT(src_path)

Input

  • src_path path to the image, containing ground-truth disparity map.

Output

  • dst output disparity map, int16 depth.

Supports basic Middlebury and MPI-Sintel formats. Note that the resulting disparity map is scaled by 16.
