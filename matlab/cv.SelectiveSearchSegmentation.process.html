cv.SelectiveSearchSegmentation/process

Based on all images, graph segmentations and stragies, computes all possible rects and return them

rects = obj.process()

Output

  • rects The list of rects as a Nx4 numeric matrix [x,y,w,h; ...]. The first ones are more relevents than the lasts ones.
