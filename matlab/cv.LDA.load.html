cv.LDA/load

Deserializes this object from a given filename

lda.load(filename)
lda.load(str, 'FromString',true)

Input

  • filename name of file to load
  • str String containing serialized object you want to load.

Options

  • FromString Logical flag to indicate whether the input is a filename or a string containing the serialized object. default false
