cv.thinning

Applies a binary blob thinning operation, to achieve a skeletization of the input image

dst = cv.thinning(src)
dst = cv.thinning(src, 'OptionName',optionValue, ...)

Input

  • src Source 8-bit single-channel image, containing binary blobs, with blobs having 255 pixel values.

Output

  • dst Destination image of the same size and the same type as src.

Options

  • ThinningType Which thinning algorithm should be used. One of:
    • ZhangSuen Thinning technique of Zhang-Suen (default).
    • GuoHall Thinning technique of Guo-Hall.

The function transforms a binary blob image into a skeletized form using the technique of Zhang-Suen.
