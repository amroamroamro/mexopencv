cv.BinaryDescriptor/BinaryDescriptor

Create a BinaryDescriptor object with default parameters (or with the ones provided)

obj = cv.BinaryDescriptor()
obj = cv.BinaryDescriptor('OptionName',optionValue, ...)

Options

  • KSize the size of Gaussian kernel: ksize-by-ksize. default 5
  • NumOfOctave the number of image octaves. default 1
  • ReductionRatio image's reduction ratio in construction of Gaussian pyramids. default 2
  • WidthOfBand the width of band; default 7

If no argument is provided, constructor sets default values. Default values are strongly recommended.
