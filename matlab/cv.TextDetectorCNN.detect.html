cv.TextDetectorCNN/detect

Detect text inside an image

[bbox, conf] = obj.detect(img)

Input

  • img an image to process, expected to be a 8-bit color image of any size.

Output

  • bbox a vector of rectangles of the detected word bounding boxes.
  • conf a vector of float of the confidences the classifier has for the corresponding bounding boxes.
