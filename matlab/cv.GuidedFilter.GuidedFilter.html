cv.GuidedFilter.guidedFilter

Simple one-line Guided Filter call

dst = cv.GuidedFilter.guidedFilter(src, guide)
dst = cv.GuidedFilter.guidedFilter(src, guide, 'OptionName',optionValue, ...)

Input

  • src filtering image with any numbers of channels.
  • guide guided image with up to 3 channels, if it have more then 3 channels then only first 3 channels will be used.

Output

  • dst output image.

Options

  • Radius radius of Guided Filter. default 7
  • EPS regularization term of Guided Filter. eps^2 is similar to the sigma in the color space into cv.bilateralFilter. default 500.0
  • DDepth optional depth of the output image. DDepth can be set to -1, which will be equivalent to class(src). Default -1

If you have multiple images to filter with the same guided image then use GuidedFilter interface to avoid extra computations on initialization stage.
