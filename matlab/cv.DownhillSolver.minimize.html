minimize (cv.DownhillSolver) - MATLAB File Help
cv.DownhillSolver/minimize

Runs the algorithm and performs the minimization

[x,fx] = solver.minimize(x0)

Input

  • x0 The initial point, that will become a centroid of an initial simplex.

Output

  • x After the algorithm will terminate, it will be set to the point where the algorithm stops, the point of possible minimum.
  • fx The value of a function at the point found.

The sole input parameter determines the centroid of the starting simplex (roughly, it tells where to start), all the others (terminal criteria, initial step, function to be minimized) are supposed to be set via the setters before the call to this method or the default values (not always sensible) will be used.
