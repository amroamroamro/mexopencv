cv.StereoSGBM/Mode

Algorithm Mode, default 'SGBM'

  • SGBM 5-directional version of the algorithm
  • HH 8-dir mode, runs the full-scale two-pass dynamic programming algorithm (slowest, memory intensive)
  • SGBM3Way 3-dir mode (fastest)
  • HH4 4-directional variation of SGBM
