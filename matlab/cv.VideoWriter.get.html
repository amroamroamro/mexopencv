cv.VideoWriter/get

Returns the specified VideoWriter property

value = vid.get(prop)

Input

  • prop Property identifier. It can be specified as a string (one of the recognized properties), or directly as its corresponding integer code.

Output

  • value Value for the specified property (as a double). Value 0 is returned when querying a property that is not supported by the backend used by the VideoWriter instance.
