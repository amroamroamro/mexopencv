cv.LATCH

Class for computing the LATCH descriptor

LATCH is a binary descriptor based on learned comparisons of triplets of image patches.

Note: the descriptor can be coupled with any keypoint extractor. The only demand is that if you use set RotationInvariance = True then you will have to use an extractor which estimates the patch orientation (in degrees). Examples for such extractors are cv.ORB and cv.SIFT.
