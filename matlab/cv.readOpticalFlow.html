cv.readOpticalFlow

Read a .flo file

flow = cv.readOpticalFlow(path)

Input

  • path Path to the file to be loaded.

Output

  • flow Flow field of single floating-point type, 2-channel. First channel corresponds to the flow in the horizontal direction (u), second the vertical (v).

Function for reading .flo files in "Middlebury" format, see: Middlebury

The function cv.readOpticalFlow loads a flow field from a file and returns it as a single matrix.
