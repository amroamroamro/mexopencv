cv.RotationWarper/warpBackward

Projects the image backward

dst = obj.warpBackward(src, K, R, dst_size)
dst = obj.warpBackward(src, K, R, dst_size, 'OptionName',optionValue, ...)

Input

  • src Projected image.
  • K Camera intrinsic parameters, 3x3 single matrix.
  • R Camera rotation matrix, 3x3 single matrix.
  • dst_size Backward-projected image size [w,h].

Output

  • dst Backward-projected image.

Options

  • InterpMode Interpolation mode, see cv.remap. default 'Linear'
  • BorderMode Border extrapolation mode, see cv.remap. default 'Constant'
