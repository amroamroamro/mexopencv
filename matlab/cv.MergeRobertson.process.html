cv.MergeRobertson/process

Merges images

dst = obj.process(src, etimes)
dst = obj.process(src, etimes, response)

Input

  • src vector of input images, all of the same size and uint8 type.
  • etimes vector of exposure time values for each image.
  • response 256x1xCN single matrix with inverse camera response function (CRF) for each pixel value, it should have the same number of channels as images src{i}.

Output

  • dst result image, same size as src{i} and single type.

The function has a short version, that doesn't take the extra response argument.
