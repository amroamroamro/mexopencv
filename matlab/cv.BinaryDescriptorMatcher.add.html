cv.BinaryDescriptorMatcher/add

Store locally new descriptors to be inserted in dataset, without updating dataset

matcher.add(descriptors)

Input

  • descriptors cell array of matrices containing descriptors to be inserted into dataset. Each matrix descriptors{i} should contain descriptors relative to lines extracted from i-th image.
