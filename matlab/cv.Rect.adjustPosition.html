cv.Rect.adjustPosition

Shift a rectangle by a certain offset

r = cv.Rect.adjustPosition(r, pt)

Input

  • r input rectangle [x,y,w,h].
  • pt point [ptx,pty].

Output

  • r output rectangle [x+ptx,y+pty,w,h].
