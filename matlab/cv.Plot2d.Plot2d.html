cv.Plot2d/Plot2d

Creates Plot2d object

obj = cv.Plot2d(dataY)
obj = cv.Plot2d(dataX, dataY)

Input

  • dataY 1xN or Nx1 matrix containing Y values of points to plot. In the first variant, X values will be equal to indexes of corresponding elements in data matrix, i.e x = 0:(numel(y)-1).
  • dataX 1xN or Nx1 matrix, X values of points to plot.
