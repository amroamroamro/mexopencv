cv.DescriptorMatcher/add

Adds descriptors to train a descriptor collection

matcher.add(descriptors)

If the collection is not empty, the new descriptors are added to existing train descriptors.

Input

  • descriptors Descriptors to add. Each descriptors{i} is a set of descriptors from the same train image. Can be either a matrix or a cell array of matrices (matrices of type uint8 or single)
