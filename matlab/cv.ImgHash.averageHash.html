cv.ImgHash.averageHash

Calculates the average hash in one call

hash = cv.ImgHash.averageHash(img)

Input

  • img input image want to compute hash value, type should be uint8 with 1/3/4 channels.

Output

  • hash Hash value of input, it will contain 16 hex decimal number, return type is uint8
