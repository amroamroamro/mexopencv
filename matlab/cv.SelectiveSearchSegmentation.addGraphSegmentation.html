cv.SelectiveSearchSegmentation/addGraphSegmentation

Add a new graph segmentation in the list of graph segementations to process

obj.addGraphSegmentation()
obj.addGraphSegmentation('OptionName',optionValue, ...)

Options

  • Sigma The sigma parameter, used to smooth image. default 0.5
  • K The k parameter of the algorithm. default 300
  • MinSize The minimum size of segments. default 100
