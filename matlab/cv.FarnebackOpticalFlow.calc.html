cv.FarnebackOpticalFlow/calc

Calculates an optical flow

flow = obj.calc(I0, I1)
flow = obj.calc(I0, I1, 'OptionName',optionValue, ...)

Input

  • I0 first 8-bit single-channel input image.
  • I1 second input image of the same size and the same type as I0.

Output

  • flow computed flow image that has the same size as I0 and type single (2-channels).

Options

  • InitialFlow specify the initial flow. Not set by default.
