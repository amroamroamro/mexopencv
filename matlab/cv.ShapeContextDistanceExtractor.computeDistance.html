computeDistance (cv.ShapeContextDistanceExtractor) - MATLAB File Help
cv.ShapeContextDistanceExtractor/computeDistance

Compute the shape distance between two shapes defined by its contours

dist = obj.computeDistance(contour1, contour2)

Options

  • contour1 Contour defining first shape. A numeric Nx2/Nx1x2/1xNx2 array or a cell-array of 2D points {[x,y], ...}
  • contour2 Contour defining second shape. Same format as contours1.

Output

  • dist output distance.
