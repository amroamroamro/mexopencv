cv.ImgHash.blockMeanHash

Computes block mean hash of the input image

hash = cv.ImgHash.blockMeanHash(img)
hash = cv.ImgHash.blockMeanHash(img, 'OptionName',optionValue, ...)

Input

  • img input image want to compute hash value, type should be uint8 with 1/3/4 channels.

Output

  • hash Hash value of input, it will contain 16 hex decimal number, return type is uint8.

Options

  • Mode block mean hash mode, one of:
    • Mode0 (default) use fewer blocks and generates 16*16/8 uint8 hash values.
    • Mode1 use block blocks (step_sizes/2) and generates fix(31*31/8)+1 uint8 hash values.
