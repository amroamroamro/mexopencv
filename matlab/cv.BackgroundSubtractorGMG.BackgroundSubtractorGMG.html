cv.BackgroundSubtractorGMG/BackgroundSubtractorGMG

Creates a GMG Background Subtractor

bs = cv.BackgroundSubtractorGMG()
bs = cv.BackgroundSubtractorGMG('OptionName', optionValue, ...)

Options

  • InitializationFrames number of frames used to initialize the background models. default 120
  • DecisionThreshold Threshold value, above which it is marked foreground, else background. default 0.8

Default constructor sets all parameters to default values.
